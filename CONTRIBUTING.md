# Contributing

This is a personal repository.
It is public so you can read it, learn from it, and fork it under the terms of the MIT license.

Contributions are not accepted.
Only the owner (@atqamz) commits and merges changes.
Pull requests will be closed unopened; do not send them.

Found a bug, a mistake, or have a question?
Open an issue.

Want your own version?
Fork it. The MIT license is the only condition.
